windows基础黑客常用dos命令

系统目录

  1. windows
  2. program files
  3. 用户
  4. perflogs
  5. perflogs是windows7的日志信息，如磁盘扫描 错误信息，删掉可以，但不建议删，删掉反而会降低系统速度，Perflogs是系统自动生成的。

服务

  1. 服务是一种应用程序类型，它在后台运行。
  2. 服务应用程序通常可以在本地和通过网络为用户提供一些功能，列如客户端/服务器应用程序、Web服务器、数据库服务器以及其他基于服务器的应用程序
  3. 运行里面输入services.msc快速打开
  4. 常见服务：web服务、dns服务、dhcp服务、邮件服务、telnet服务、ssh服务、ftp服务、smb服务
  5. 所有的服务都对应一个或多个端口，但端口只对应一个服务
  6. 常见端口：

    HTTP协议代理服务器常用端口号：80/8080/3128/8081/9080

    FTP(文件传输)协议代理服务器常用端口号：21

    Telnet（远程登录）协议代理服务器常用端口：23

    TFTP：23

    SSH（安全登录）、SCP(文件传输)、端口重定向：22

    SMTP：25

    TOMCAT：8080

    WIN2003远程登录，默认的端口号为：3389

    MS SQL SERVER数据库：1433

黑客通过端口可以干什么？

  1. 信息搜集
  2. 目标探测
  3. 服务判断
  4. 系统判断
  5. 系统角色分析

注册表的作用

  1. 注册表是windows操作系统中的一个核心数据库，其中存放着各种参数，直接控制着windows的启动、硬件驱动程序的装载以及一些windows应用程序的运行，从而在整个系统中起着核心作用。这些作用包括了软、硬件的相关配置和状态信息，比如注册表中保存有应用程序和资源管理器外壳的初始条件、首选项和卸载数据等，联网计算机的整个系统的设置和各种许可，文件扩展名与应用程序的关联，硬件部件的描述、状态、和属性，性能记录和其他底层的系统状态信息，以及其他数据等。
  2. 运行里面输入regedit快速打开
  3. 注册表结构
    • HKEY_CLASSES_ROOT 管理文件系统。
    • HKEY_CURRENT_USER 管理系统当前的用户信息。
    • HKEY_LOCAL_MACHINE 管理当前系统硬件配置。 这个跟键里面的许多子键与System.ini文件中设置项类似。
    • HKEY_USERS 管理系统的用户信息
    • HKEY_CURRENT_CONFIG 管理当前用户的系统配置。

黑客常用的DOS命令

  1. color 改变cmd颜色
  2. ping -t -l 65550 ip 死亡之ping（发送大于64k的文件并一直ping死亡之ping）
  3. ipconfig 查看ip
  4. ipconfig /release 释放ip
  5. ipconfig /renew 重新获得ip
  6. systeminfo 查看系统信息
  7. arp -a
  8. net view 查看局域网内其他计算机名称
  9. shutdown -s -t 180 -c “你被黑了，系统马上关机”
  10. shutdown -a 取消
  11. dir 查看目录
  12. cd 切换目录
  13. start www.baidu.com 打开网页
  14. start 123.txt 打开123.txt
  15. copy con c:\123.txt 创建123.txt文件

    hello baidu

    ctrl+z 回车
  16. md 目录名 创建目录名
  17. rd 123 删除文件夹
  18. ren 原文件名 新文件名 重命名文件名
  19. del 删除文件
  20. copy 复制文件
  21. move 移动文件
  22. tree 树形列出文件夹结构
  23. telnet
  24. net use k:\192.168.1.1\c$
  25. net use k:\192.168.1.1\c$/del
  26. net start查看开启了哪些服务
  27. net start 服务台 开启服务：（如：net start telnet，net start schedule）
  28. net stop 服务名 停止某服务
  29. netstat -a 查看开启了哪些端口，常用netstat -an
  30. netstat -n 查看端口的网络连接情况
  31. netstat -v 查看正在进行的工作
  32. netsh 网络工具
  33. at id号 开启已注册的某个计划任务
  34. at/delete 停止所有计划任务，用参数/yes则不需要确认就直接停止
  35. at id号 /delete停止某个已注册的计划任务
  36. at 查看所有的计划任务
  37. attrib 文件名（目录名）查看某文件（目录）的属性
  38. attrib 文件名 -A -R -S -H或+A +R +S +H 去掉（添加）某文件的存档，只读，系统，隐藏 属性：用+则是添加为某属性
  39. net user 用户名 密码 /add 建立用户
  40. net user guest/active:yes 激活guest用户
  41. net user 查看有哪些用户
  42. net user 账户名 查看账户的属性
  43. net localgroup administrators 用户名 /add 把“用户”添加到管理员中使其具有管理员权限，注意： administrator后加s用复数
  44. net user guest 12345 用guest用户登陆后将密码改为12345
  45. net user user password 更改系统登陆密码
  46. net share 查看本地开启的共享
  47. net share ipc$ 开启ipc$共享
  48. net share ipc$ /del删除ipc$共享
  49. net share c$ /del 删除C: 共享
  50. tasklist 查看进程
  51. taskkill pid 结束进程
  52. msg user 聊天内容 用户会出现一个弹框
